LifeHub

A personal productivity and lifestyle management web app built with React and Vite.
LifeHub helps you organize your daily life with a clean dashboard and modular design.

 Features
•	Modern, responsive UI with Tailwind CSS
•	Dashboard and Home pages with routing setup
•	Component-based architecture for scalability
•	Fast development environment using Vite
•	Easy to customize and extend

 Tech Stack
•	React
•	Vite
•	Tailwind CSS
•	JavaScript (ES6+)

 Setup
# Clone the repository
git clone https://github.com/<your-username>/lifehub.git
cd lifehub

# Install dependencies
npm install

# Start the development server
npm run dev
