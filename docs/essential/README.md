# 📚 Essential Documentation

This folder contains documentation **required to run and use the LifeHub project**.

## Files in This Folder

### Setup & Configuration
- **POSTGRESQL_SETUP.md** - Complete PostgreSQL database setup guide
- **QUICK_START_POSTGRESQL.md** - Quick reference for database setup
- **backend-README.md** - Backend overview, setup, and basic API documentation

### Running the Application
- **RUN_BACKEND.md** - Detailed instructions for running the Spring Boot backend

### API Reference
- **API_DOCUMENTATION.md** - Complete API endpoint documentation with examples

### Optional Features
- **REDIS_CACHE_TESTING.md** - Redis caching setup and testing guide (optional feature)

## Quick Start Guide

1. **Database Setup**: Read `POSTGRESQL_SETUP.md` or `QUICK_START_POSTGRESQL.md`
2. **Backend Setup**: Read `backend-README.md` for overview, then `RUN_BACKEND.md` for detailed steps
3. **Frontend Setup**: Navigate to `../../frontend/` and follow `README.md`
4. **API Usage**: Refer to `API_DOCUMENTATION.md` for all available endpoints
5. **Optional**: Set up Redis using `REDIS_CACHE_TESTING.md` for caching
